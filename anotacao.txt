passo a passo para enviar um commit para o repositório com comandos

primeiro - git status (verifica se tem algo para salvar)
segundo - git add arquivo (adiciona arquivo)
terceiro -git commit -m (nome do arquivo entre aspas)
quarto - git push (manda pra nuvem)

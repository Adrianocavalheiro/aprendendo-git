entrar na pasta = cd (comando para entrar na pasta)

Ver conteudo da pasta
ls (comando para ver o conteúdo da pasta)

passo a passo para enviar um commit para o repositório com comandos

primeiro -git commit -m (nome do arquivo entre aspas)
segundo - git status (verifica se tem algo para salvar)
terceiro - git add arquivo (adiciona arquivo)
quarto - git push (manda pra nuvem)
